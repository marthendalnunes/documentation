The Starter Template is a pre-structured blueprint designed to streamline the creation of new integrations within the TurboETH ecosystem.

# Integration

- Demo: [turboeth.xyz/integration/starter](https://www.turboeth.xyz/integration/starter)
- Code: [github.com/turbo-eth/template-web3-app/tree/integrations/integrations/starter](https://github.com/turbo-eth/template-web3-app/tree/integrations/integrations/starter)

**Folder Structure:**
```
integrations/starter
├─ abis/
│  ├─ starter-abi.ts
├─ client/

├─ components/

├─ generated/

├─ hooks/

├─ utils/

├─ index.ts
├─ README.md
```
