TurboETH is a Web3 App Template designed with interopability in mind.

View the `integrations` branch to see all of the current integrations.

[TurboETH Integrations Branch](https://github.com/turbo-eth/template-web3-app/tree/integrations)

- [Sign-In With Ethereum](https://github.com/turbo-eth/template-web3-app/tree/integrations/integrations/siwe)

## Integration Types

- **Blockchain** - smart contract read, write and event utilities.
- **API** - Web3 service: caching, identity, storage, etc...

### Blockchain Integration

```
integrations/[smart-contract]
├─ abis/
├─ components/
├─ hooks/
├─ erc20-wagmi.ts
├─ wagmi.config.ts
├─ README.md
```

**ERC20 integration example:**
```
integrations/erc20
├─ abis/
│  ├─ erc20-abi.ts
│  ├─ erc20-bytecode.ts
│  ├─ erc20-mintable-abi.ts
│  ├─ erc20-mintable-bytecode.ts
├─ components/
│  ├─ branch-token-minted.tsx
│  ├─ erc20-deploy.tsx
│  ├─ erc20-event-mint.tsx
│  ├─ erc20-event-transfer.tsx
│  ├─ erc20-read.tsx
│  ├─ erc20-write-mint.tsx
│  ├─ erc20-write-transfer.tsx
├─ hooks/
│  ├─ use-token-storage.ts
├─ erc20-wagmi.ts
├─ wagmi.config.ts
├─ README.md
```

### API Service Integration

```
integrations/[api-service]
├─ actions/
├─ components/
├─ hooks/
├─ routes/
├─ client.ts
├─ README.md
```

**Disco integration example:**

```
integrations/disco
├─ actions/
│  ├─ get-credentials-from-did.ts
│  ├─ get-profile-from-address-simple.ts
│  ├─ get-profile-from-address.ts
├─ components/
│  ├─ disco-profile-basic.tsx
│  ├─ disco-profile-credentials.tsx
├─ hooks/
│  ├─ use-disco-get-credentials-from-did.ts
│  ├─ use-disco-get-profile-from-address.ts
│  ├─ use-disco-get-profile-from-did.ts
├─ routes/
│  ├─ route-credentials-from-did.ts
│  ├─ route-disco-profile-from-address.ts
│  ├─ route-disco-profile-from-did.ts
├─ disco-client.ts
├─ README.md
```
