## 1-Click Deploy
TurboETH application templates can be deployed in 1 click using Vercel.

Visit [TurboEth.xyz](https://www.turboeth.xyz/) to get started

## Command Line Interface

The `pnpm` CLI is the recommended package manager but `npm` and `yarn` should work too.

```bash
# SSH
git clone git@github.com:turbo-eth/template-web3-app.git

# HTTPS
git clone https://github.com/turbo-eth/template-web3-app.git
```

The `pnpm` CLI is the recommended package manager but `npm` and `yarn` should work too.

```bash
pnpm install
```

#### Development
```bash
pnpm dev
```

#### Build
```bash
pnpm dev
```
